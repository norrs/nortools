# Desktop Troubleshooting

## App Does Not Start

Check that you downloaded the package for the correct operating system and CPU architecture.

## Network Checks Fail

Some tools contact public DNS, HTTP, SMTP, or local discovery services. Firewalls, VPNs, and captive portals can affect results.

## Screenshots Differ From The Docs

The docs track released screenshots. If you are running a development build, the UI may be newer than the published documentation.
